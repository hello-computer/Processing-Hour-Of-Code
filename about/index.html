О проекте

Эти уроки были сделаны огранизацией Processing Foundation в рамках проекта Hour of Code™ — общенациональной инициативы проекта Code.org по знакомству студентов с программированием. Программа проходила в рамках недели Computer Science Education Week с 9 по 15 декабря 2013 года. Ее целью было дать возможность миллионам студентов познакомиться с программированием в качетсве способа мышления и творчества.

Наш вклад в этот проект посвящен языку Processing, который создан специально для того чтобы познакомить сообщество художников с программированием. Именно поэтому Processing активно используется для обучения основам программирования в контексте визуального искусства. Processing является программным обеспечением с открытым исходным кодом и распространяется абсолютно бесплатно. Его можно скачать на сайте Processing и здесь же найти более подробную информацию.

Технологии

Сайт сделан на основе Jekyll и размещен на хостинге GitHub Pages. Возможность поделится своим кодом сделана с помощью сервиса GitHub Gist, а видео размещены на Vimeo. Основа сайта была автоматически создана с помощью генератора шаблонов Initialzr, который использует библиотеки HTML5 Boilerplate, Bootstrap и Modernizr.

Сайт в значительной степени использует открытые Javascript-библиотеки, а именно: Ace для редактирования кода, Popcorn.js для поддержки событий видео и Processing.js для запсука скетчей на Processing прямо в браузере. Также в проекте используются библиотеки: jQuery, Spectrum, FileSaver.js и Bootstrap Hover Dropdown Plugin.

Русская версия

Русская версия проекта создана студией Hello Computer в целях популяризации Processing и творческого программирования в России. Видеоролики перевел Иван Фролкин, озвучила Аня Кулысова, сайт перевел и адаптировал Максим Яхонтов.
